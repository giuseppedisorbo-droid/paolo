Peppe-System: V2 Database Importer

Questo script si connetterà al tuo Firebase (utilizzando le API Key del progetto Peppe AI Platform) e importerà le 12 collezioni come richiesto nell'architettura V2.

ATTENZIONE: I documenti con lo stesso ID veranno sovrascritti. L'operazione non elimina i vecchi documenti con ID diversi, ma scriverà esattamente i dati di seed.
